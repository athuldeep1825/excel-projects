Project Name: Bike Sales Report

This project analyzes bike sales data using Microsoft Excel to uncover sales trends and customer purchasing behavior. The dataset was cleaned and structured to ensure accuracy and consistency before analysis.

Key Features:

Data cleaning and preprocessing in Excel

Sales analysis using Pivot Tables

Interactive dashboard creation

Filters and slicers for dynamic data exploration

Tools Used:

Microsoft Excel

Outcome:

The dashboard provides clear insights into bike sales performance, customer demographics, and purchasing patterns, supporting data-driven business decisions.
